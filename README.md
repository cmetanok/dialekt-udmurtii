# Диалектологическая карта Удмуртии

Интерактивная карта русских говоров Удмуртии с данными из программы ДАРЯ.

## Возможности
- Просмотр населенных пунктов на карте
- Фильтрация по диалектным вопросам
- Режим редактирования с автоматическим определением координат
- Экспорт данных в CSV

## Технологии
- Python + Streamlit
- Folium для карт
- Google Sheets как база данных
